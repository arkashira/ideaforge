# Ideaforge – Core Ideation Engine

`ideaforge` generates a ranked list of software‑tool ideas based on a user‑provided niche,
budget range, and technical constraints.

## Features

* Input: niche keywords, budget range, technical constraints.
* Output: at least 5 ideas, each with a one‑sentence description, target persona,
  demand score (0‑100) and a composite profitability score.
* Deterministic – the same inputs always produce the same ranked list.
* No external dependencies (standard library only).
