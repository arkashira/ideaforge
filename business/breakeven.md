# breakeven.md

## Unit Economics & Break-even Analysis for Ideaforge

### Cost per Active User
- **Compute Costs**: $0.05 per hour per user
  - Estimated usage: 10 hours/month
  - Monthly compute cost per user: $0.05 * 10 = **$0.50**

- **Storage Costs**: $0.02 per GB per month
  - Estimated storage per user: 1 GB
  - Monthly storage cost per user: $0.02 * 1 = **$0.02**

- **Bandwidth Costs**: $0.01 per GB
  - Estimated bandwidth usage per user: 5 GB/month
  - Monthly bandwidth cost per user: $0.01 * 5 = **$0.05**

- **Total Cost per Active User**: 
  - Compute + Storage + Bandwidth = $0.50 + $0.02 + $0.05 = **$0.57**

### Pricing Tiers
1. **Basic Tier**: $10/month
   - Features: Idea generation tool, Basic validation metrics, Community access

2. **Pro Tier**: $25/month
   - Features: All Basic features, Advanced validation metrics, Collaboration tools, Priority support

3. **Premium Tier**: $50/month
   - Features: All Pro features, Custom idea reports, One-on-one consultation, Exclusive content

### Customer Acquisition Cost (CAC) Range
- Estimated CAC: $30 - $50 per user
  - Based on marketing spend, outreach, and conversion rates.

### Lifetime Value (LTV) Estimate
- Average user lifespan: 24 months
- Average revenue per user (ARPU) across tiers: 
  - (10 + 25 + 50) / 3 = $28.33
- LTV = ARPU * Average user lifespan = $28.33 * 24 = **$680**

### Break-even Users Count
- Monthly fixed costs (marketing, salaries, infrastructure): $5,000
- Contribution margin per user (ARPU - Cost per user):
  - Average ARPU: $28.33
  - Cost per user: $0.57
  - Contribution margin: $28.33 - $0.57 = **$27.76**
  
- Break-even users count = Fixed Costs / Contribution Margin
  - Break-even users count = $5,000 / $27.76 ≈ **180 users**

### Path to $10K MRR
- Target MRR: $10,000
- Average Revenue per User (ARPU) across tiers: $28.33

- Required users to reach $10K MRR = $10,000 / $28.33 ≈ **353 users**

#### Tier Distribution for $10K MRR
- Assuming a mix of tiers:
  - Basic Tier: 100 users * $10 = $1,000
  - Pro Tier: 150 users * $25 = $3,750
  - Premium Tier: 103 users * $50 = $5,150

- Total MRR from this distribution = $1,000 + $3,750 + $5,150 = **$9,900** (close to $10K)

- Adjusting to reach $10K:
  - Increase Basic Tier to 110 users: 110 * $10 = $1,100
  - Total MRR = $1,100 + $3,750 + $5,150 = **$10,000**

### Summary
- **Total Cost per Active User**: $0.57
- **Pricing Tiers**: Basic ($10), Pro ($25), Premium ($50)
- **CAC Range**: $30 - $50
- **LTV Estimate**: $680
- **Break-even Users Count**: 180 users
- **Path to $10K MRR**: 110 Basic, 150 Pro, 103 Premium users.
